# EasyTrip BD 
**EasyTrip BD** is a web-based tourism and hotel booking platform focused on Bangladesh. The platform helps travelers discover tourist destinations, explore hotels, view reviews, and book accommodations. Hotel owners can create and manage hotel listings, rooms, and booking information through a dedicated dashboard.
The goal of the project is to provide a centralized platform where travelers can easily plan trips within Bangladesh while hotel owners can promote and manage their accommodations.

## Repository Purpose
This repository demonstrates a complete **Software Development Life Cycle (SDLC)** and Requirements Engineering journey:

Problem Discovery -> Requirement Elicitation -> Product Requirements Document (PRD) -> User Stories -> Requirements -> Software Requirements Specification (SRS) -> Design -> Technical Design Document (TDD) -> Database -> Application Programming Interface (API) -> Testing -> Release
