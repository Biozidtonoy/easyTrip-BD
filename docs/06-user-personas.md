# Bangladesh Tourism Platform – User Personas

## Persona P1 - Arif (Domestic Traveler)

| Attribute        | Details                                                                                            |
| ---------------- | -------------------------------------------------------------------------------------------------- |
| Demographics     | 24, university student who enjoys traveling during semester breaks                                 |
| Goals            | Discover new tourist destinations and plan affordable trips                                        |
| Motivations      | Explore Bangladesh, save time planning trips, and read authentic travel experiences                |
| Pain Points      | Tourism information is scattered across different websites and social media platforms              |
| Technology Usage | Mobile-first, occasionally uses laptop for trip planning                                           |
| Typical Workflow | Search destinations → Compare hotels → Read reviews → Book accommodation → Share travel experience |

---

## Persona P2 - Nusrat (Family Traveler)

| Attribute        | Details                                                                            |
| ---------------- | ---------------------------------------------------------------------------------- |
| Demographics     | 35, working professional planning family vacations                                 |
| Goals            | Find family-friendly destinations and reliable accommodations                      |
| Motivations      | Safe travel, comfortable hotels, and trustworthy reviews                           |
| Pain Points      | Difficulty comparing accommodations and verifying hotel quality                    |
| Technology Usage | Desktop for planning, mobile during travel                                         |
| Typical Workflow | Browse destinations → Filter hotels → Check ratings → Book hotel → Manage bookings |

---

## Persona P3 - Rahim (Hotel Owner)

| Attribute        | Details                                                                                                  |
| ---------------- | -------------------------------------------------------------------------------------------------------- |
| Demographics     | 42, owner of a local resort in Cox's Bazar                                                               |
| Goals            | Promote accommodations, manage rooms, and increase bookings                                              |
| Motivations      | Reach more travelers and improve business visibility                                                     |
| Pain Points      | Limited online exposure and difficulty managing booking requests                                         |
| Technology Usage | Desktop browser for business management                                                                  |
| Typical Workflow | Login → Manage hotel profile → Update room availability → Review bookings → Respond to customer feedback |

---

## Persona P4 - Sumi (Tourism Business Owner)

| Attribute        | Details                                                                     |
| ---------------- | --------------------------------------------------------------------------- |
| Demographics     | 31, owner of a local tour agency                                            |
| Goals            | Promote tourism services and attract more customers                         |
| Motivations      | Increase business opportunities through online visibility                   |
| Pain Points      | Marketing relies heavily on social media and word-of-mouth                  |
| Technology Usage | Desktop and mobile browser                                                  |
| Typical Workflow | Update business information → Promote services → Monitor customer inquiries |

---

## Persona P5 - Hasan (Platform Administrator)

| Attribute        | Details                                                                                                     |
| ---------------- | ----------------------------------------------------------------------------------------------------------- |
| Demographics     | 30, system administrator responsible for platform operations                                                |
| Goals            | Ensure platform quality, approve content, and maintain security                                             |
| Motivations      | Reliable platform management and high user satisfaction                                                     |
| Pain Points      | Managing inappropriate content, fake listings, and user disputes                                            |
| Technology Usage | Desktop browser with administrative dashboard                                                               |
| Typical Workflow | Review hotel submissions → Approve destinations → Moderate reviews → Manage users → Monitor system activity |

---

## Persona Coverage to Requirement Themes

| Theme                      | Personas     | Related Features                                                           |
| -------------------------- | ------------ | -------------------------------------------------------------------------- |
| Destination Discovery      | P1, P2       | Destination browsing, search, filtering, destination details               |
| Hotel Booking              | P1, P2       | Hotel listings, room availability, booking management                      |
| Hotel Management           | P3           | Hotel dashboard, room management, booking management                       |
| Tourism Business Promotion | P4           | Business profile management and service promotion                          |
| Platform Administration    | P5           | User management, hotel approval, destination management, review moderation |
| Security & Access Control  | All Personas | Authentication, role-based authorization, secure account management        |
